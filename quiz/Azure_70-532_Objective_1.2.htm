Both Connection Strings and App Settings can be modified when accessed from application code?
True
False
App Settings and Connection Strings are read-only when accessed from application code. To modify their values, you must either use the Management Portal, Azure Management API, Windows Powershell, or xplat-cli.
What are the steps for defining an App Setting in the existing portal?
Navigate to the management portal at https://manage.windowsazure.com
Navigate to the management portal at https://portal.azure.com
Navigate to the "Configure" tab of the Azure Web App
Scroll down to the "app settings" section
Modify the JSON that defines the app settings
Modify the XML that defines the app settings
Type the name for the app setting in the "Key" field
Type the value for the app setting in the "Value" field
