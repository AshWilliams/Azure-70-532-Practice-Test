What are the steps for creating a Windows Virtual Machine within the Existing Portal?
Navigate to the management portal at https://manage.windowsazure.com
Navigate to the management portal at https://portal.azure.com
Click "New" in the Command Bar
Select "Compute", "Virtual Machine", then "From Gallery"
Select "Compute", "Virtual Machine", then "Windows Server 2012"
Select the Windows Virtual Machine Image to use
Select the Computer Name, Size and Admin username/password
Select the resources for Networking, Storage and Availability
Select the VM Agent and other available extensions
Source: http://azure.microsoft.com/en-us/documentation/articles/virtual-machines-windows-create-custom/
What are the steps for creating a Windows Virtual Machine within the Preview Portal?
Navigate to the management portal at https://manage.windowsazure.com
Navigate to the management portal at https://portal.azure.com
Click "New", the "Compute"
Click "Azure Marketplace" to open up the Marketplace blade
Scroll down to the "Virtual Machines" section and select "Windows Server"
Select the version of Windows Server desired and click "Create"
Enter the VM settings and click "Create"
What does the VM Agent provide?
Server resource management monitoring
Automatic installation of Windows Updates
Environment for extensions to be installed
The VM Agent provides an environment for extensions to be installed that can help with interacting or managing the virtual machine.
